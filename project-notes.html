Case Study: Ideas on the Go

note-taking app

User Interface design for a native note-taking app pitch presentation.

Quicklinks: Project Brief | Objective | Wireframes | Final Concept |

project brief:

Develop the concept of a "less but better" note-taking app for a preliminary pitch to investors.

objective:

Create high-fidelity wireframes for 5-7 mobile screens in the program Figma, designed be clean, simple & easy-to-use.

high-fidelity wireframes:

Each UI element is defined by a "less is more" function combined with a bold "more is more" maximalism color scheme. The User Experience allows for easy editing, reading and organizing of notes.

final concept:

With this potential app, the user can quickly jot down basic notes, thoughts or reminders as soon as they occur. They can make lists, plan events and upload images all on the go!
